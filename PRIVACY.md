# Política de Privacidade — YouTube Foco

**Data de vigência:** 19 de janeiro de 2026

A extensão **YouTube Foco** foi criada para reduzir distrações no YouTube. Esta política descreve como lidamos com dados.

## Coleta de dados
O **YouTube Foco não coleta, vende ou compartilha** dados pessoais.

## Uso e armazenamento local
As preferências da extensão (por exemplo: modo foco ligado/desligado, perfil selecionado e regras) são armazenadas **localmente no seu navegador** usando `chrome.storage.local`.

## Compartilhamento com terceiros
Nenhum dado é enviado para servidores externos e **não há compartilhamento** com terceiros.

## Permissões
A extensão usa apenas as permissões necessárias para o seu funcionamento:
- `storage`: salvar suas preferências localmente.
- `alarms`: implementar a funcionalidade de **soneca** (reabilitar automaticamente o modo foco após um tempo).
- `https://www.youtube.com/*` (host permission): aplicar as mudanças de interface **somente no YouTube**.
